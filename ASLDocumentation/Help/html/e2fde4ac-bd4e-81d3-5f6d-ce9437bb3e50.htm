ARWorldOriginHelperRaycast Method

Performs a raycast against physical objects being tracked by ARCore. This function wraps Frame.Raycast to add the necessary offset if the WorldOrigin is moved when a Cloud Anchor is placed. Output the closest hit from the camera. Note that the Unity's screen coordinate (0, 0) starts from bottom left.
